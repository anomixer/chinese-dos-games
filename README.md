# 🎮 中文 DOS 遊戲

網址： https://dos.lol


中文 DOS 遊戲合集，目前共有 1898 款遊戲。

## 下載遊戲檔

在根目錄下運行 Python 3 腳本

``` python
python download_data.py
```

若下載出錯請參見 [Issue #26](https://github.com/rwv/chinese-dos-games/issues/26)
